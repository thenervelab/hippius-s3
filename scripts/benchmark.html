S3 Benchmarks

Status Quickstart →

Throughput benchmarks for our two supported regions: eu-central-1.hippius.com (also reachable as s3.hippius.com) and us-east-1.hippius.com. Each datapoint is one run, recording min/max/avg over 3 trials.

See https://docs.hippius.com/use/quickstart to try out Hippius S3 and store your first file.

                All benchmarks use 2 GB test files via a single PUT request (no multipart upload). Each run records 3 trials. All timestamps are in UTC. Equivalent AWS CLI invocations: 
                # Upload — single PUT, no multipart
aws s3api put-object --bucket BUCKET --key KEY --body ./file.bin \
  --endpoint-url https://eu-central-1.hippius.com

# Download
aws s3 cp s3://BUCKET/KEY ./file.bin \
  --endpoint-url https://eu-central-1.hippius.com

                Source: scripts/s3-benchmark.py in the hippius-s3 repo.
